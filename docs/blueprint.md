# **App Name**: Averzo

## Core Features:

- Admin Panel: A comprehensive admin panel to manage all aspects of the Averzo platform, including product inventory, user accounts, vendor management, sub-brands, and delivery services.
- Customer Panel: User-friendly customer panel for browsing products, placing orders, managing profiles, and tracking deliveries.
- Offline Outlet Integration: Functionality to manage offline store inventory and sales data within the same platform as the online store.
- Delivery Management: Integration with Averzo's own rider network, as well as third-party delivery services (Redx, Pathao, Paperfly, Sundarban Courier Service) to offer flexible delivery options. The LLM uses tools to monitor all active orders.
- Vendor Management: System for vendors to supply products to Averzo, including product submissions, inventory management, and payment tracking.
- Sub-Brand Management: Ability to create and manage multiple sub-brands (e.g., Aura Men) with separate product catalogs and branding.
- Product Image Management: System to host product images using placeholder images (placehold.co) for different sizes until actual images are uploaded via links or direct uploads. Products can be assigned to the main category, group and subcategory for flexible product filtering and browsing.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke trust and reliability.
- Background color: Light blue (#E0F7FA), creating a clean and airy feel.
- Accent color: A soft green (#90EE90) to indicate availability and positive interactions.
- Font stack: 'Roboto' (sans-serif) for general body text and UI elements, 'Noto Sans' (sans-serif) as a fallback for broader language support, and 'Saira' (sans-serif) for headlines and emphasized text.
- Modern, minimalist icons to represent product categories, user actions, and delivery options.
- Clean, responsive layout with a focus on visual hierarchy and ease of navigation. Utilize card-based design for product listings.
- Subtle animations for transitions, loading states, and user feedback to enhance engagement without being distracting.
